<p align="center"><h2>Closed Beta Notes</h2></p>

- Some features may be incomplete or missing.
- Expect some Z-fighting
- You are able to right click many objects in order to examine them, so be sure to press L and use your logbook if you missed something.
- After reaching the mine, feel free to keep playing to test the balance.
  Please msg me your feedback on discord afterwards. Any feedback is welcome!
- If a festive event is in progress, it may cause some thematic issues. Please go to settings > Online > Festive Content > set to off

# Instructions

## Downloading

1. Press the green "Code" button at the top right
2. Press "Download ZIP"

## Installation

1. Create a new folder anywhere (Next to your game installation is a good option)
2. Copy/paste the contents of your vanilla barony folder to it
3. Unpack all files from this archive into it, replace all files
4. Disable steam cloud for Barony if it's activated
5. Run barony.exe from the new folder
